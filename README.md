# 🩺 DiabetScan — Орнату нұсқаулығы

## Қант диабетін ерте анықтау жүйесінің веб-қосымшасы

---

## 📋 Қажетті бағдарламалар

Жобаны іске қосу үшін компьютеріңізде мыналар орнатылуы керек:

1. **Node.js** (v18 немесе одан жоғары)
2. **VS Code** немесе **IntelliJ IDEA**

---

## 🔧 1-ҚАДАМ: Node.js орнату

Node.js — бұл JavaScript-ті компьютерде іске қосуға мүмкіндік беретін платформа.

### Windows:
1. https://nodejs.org сайтына кіріңіз
2. **"LTS"** (ұсынылған нұсқа) батырмасын басыңыз — файл жүктеледі
3. Жүктелген `.msi` файлды ашыңыз
4. "Next" басып орнатыңыз (барлық баптауларды әдепкі қалдырыңыз)
5. Орнатылғанын тексеру: **Терминалды** ашып жазыңыз:
   ```
   node --version
   ```
   Нәтиже: `v18.xx.x` немесе `v20.xx.x` шығуы керек

### macOS:
1. https://nodejs.org сайтынан LTS нұсқасын жүктеңіз
2. `.pkg` файлды ашып орнатыңыз
3. Терминалда тексеріңіз: `node --version`

---

## 📁 2-ҚАДАМ: Жоба папкасын дайындау

1. `diabetscan.zip` файлды жүктеп алыңыз
2. Кез келген жерге шығарыңыз (мысалы: `C:\Users\Сіздің_атыңыз\Desktop\diabetscan`)
3. Папканың ішінде мынадай файлдар болуы керек:
   ```
   diabetscan/
   ├── index.html
   ├── package.json
   ├── vite.config.js
   └── src/
       ├── main.jsx
       └── App.jsx
   ```

---

## 💻 3-ҚАДАМ: VS Code-та ашу

### 3.1 — VS Code-ты ашыңыз
- Егер жоқ болса: https://code.visualstudio.com сайтынан жүктеп орнатыңыз

### 3.2 — Жоба папкасын ашыңыз
- VS Code → **File** → **Open Folder...** → `diabetscan` папкасын таңдаңыз

### 3.3 — Терминалды ашыңыз
- VS Code ішінде: **Terminal** → **New Terminal**
  (немесе `Ctrl + ~` пернелерін басыңыз)
- Терминал төменде ашылады

---

## 💻 3-ҚАДАМ (баламалы): IntelliJ IDEA-да ашу

### 3.1 — IntelliJ IDEA-ны ашыңыз

### 3.2 — Жобаны ашыңыз
- **File** → **Open** → `diabetscan` папкасын таңдаңыз → **OK**

### 3.3 — Терминалды ашыңыз
- Төменгі панельде **Terminal** табын басыңыз
  (немесе `Alt + F12` пернелерін басыңыз)

---

## 📦 4-ҚАДАМ: Кітапханаларды орнату (npm install)

Терминалда мына команданы жазыңыз:

```bash
npm install
```

**Не болады:**
- `package.json` файлдағы тізімге сәйкес барлық кітапханалар (React, Recharts, Vite) жүктеледі
- `node_modules` папкасы пайда болады (бұл қалыпты, өлшемі ~100-200 МБ)
- Бірнеше минут күтіңіз (интернет жылдамдығына байланысты)

**Сәтті аяқталса:**
```
added 200+ packages in 30s
```
деген жазу шығады.

**Қате шықса:**
- Интернет байланысын тексеріңіз
- `node --version` арқылы Node.js орнатылғанын тексеріңіз
- Терминалдың `diabetscan` папкасында тұрғанын тексеріңіз

---

## 🚀 5-ҚАДАМ: Жобаны іске қосу (npm run dev)

Терминалда мына команданы жазыңыз:

```bash
npm run dev
```

**Не болады:**
- Vite серверін іске қосады
- Терминалда мынадай жазу шығады:

```
  VITE v5.4.2  ready in 300 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: http://192.168.x.x:3000/
  ➜  press h + enter to show help
```

- **Браузер автоматты ашылады** http://localhost:3000 бетімен
- Егер автоматты ашылмаса — браузерді (Chrome, Edge, Firefox) ашып мекенжай жолына жазыңыз:
  ```
  http://localhost:3000
  ```

## ⏹️ 7-ҚАДАМ: Серверді тоқтату

Серверді тоқтату үшін терминалда:
- **`Ctrl + C`** пернелерін басыңыз

Қайта іске қосу үшін:
```bash
npm run dev
```

---

## 🔄 Кодты өзгерту (Hot Reload)

Vite серверін іске қосқан кезде:
- `src/App.jsx` файлын VS Code / IntelliJ-де өзгертіңіз
- **Сақтаңыз** (`Ctrl + S`)
- Браузер **автоматты жаңарады** — өзгерістер бірден көрінеді!

---

## 📦 8-ҚАДАМ: Дайын сайтты жинау (Build) — қосымша

Егер сайтты серверге жүктегіңіз келсе:

```bash
npm run build
```

Нәтижесінде `dist/` папкасы пайда болады — оны кез келген хостингке жүктеуге болады.

Дайын нәтижені алдын ала көру:
```bash
npm run preview
```

## ❓ Жиі кездесетін мәселелер

| Мәселе | Шешімі |
|--------|--------|
| `node` командасы табылмады | Node.js-ті қайта орнатыңыз, терминалды қайта ашыңыз |
| `npm install` қатесі | Интернетті тексеріңіз, `npm cache clean --force` жазып қайта орнатыңыз |
| Порт 3000 бос емес | `vite.config.js` файлында портты 3001-ге өзгертіңіз |
| Браузерде бос бет | Браузер консолін ашыңыз (F12) → қатені қараңыз |
| `node_modules` жоқ | `npm install` қайта жүргізіңіз |
